How it works

Two modes:

  1. Deterministic — If the URL has ?id=..., the site uses a hash to map that id to a stable fortune index. Each NFC tag can have its own id so the same cookie always shows the same message.
  2. Random no-repeat — With no id, the site picks fortunes randomly while avoiding repeats until it cycles through all, using localStorage.

How to program your NFC tag:

  1. Write a URL to the tag with an app like NFC Tools. Example: https://yourdomain.example/?id=COOKIE-001.
  2. Stick the tag into your piece (or glue it after firing). When someone taps, their phone opens your URL and shows the fortune.

Editing fortunes: Open fortunes.js and add or remove strings in the window.FORTUNES array. You can have 800+ fortunes.

Deploy: Host these files on Netlify, GitHub Pages, Cloudflare Pages, or any static host. Upload the folder and set your domain.
